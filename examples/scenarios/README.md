# Examples of possible interop scenarios

This folder contains examples demonstrating ESM-CJS interop scenarios [discussed](https://github.com/johnloy/esm-commonjs-interop-manual#possible-interop-scenarios-and-factors) in *EcmaScript-CommonJS Module Interop: The Missing Manual*.

- [Running a real ESM module, with only ESM dependencies in supporting Node and browsers](./esm-only/README.md)

- [Running a real ESM module in Node importing a real CJS module](./esm-importing-cjs/README.md)

- [Running a real ESM module in Node importing a transpiled, or "faux", ESM module](./esm-importing-faux/README.md)

- [Running a faux CJS module in Node, with dependencies on CJS and/or ESM modules and/or packages](./faux-in-node/README.md)

- [Running an IIFE bundle script in a browser, with dependencies on CJS and/or ESM modules and/or packages](iife-in-browser/README.md)

- [Running a real ESM bundle module in a browser with dependencies on CJS and/or ESM modules and/or packages](./esm-in-browser/README.md)
